# White Space

**White Space** is a website showcasing an art gallery focused on children’s paintings. The project aims to provide a clean, intuitive, and visually appealing experience that highlights the artwork in a simple and charming way.

---

## Getting Started

These instructions will help you run the project on your local machine for development, testing, or viewing purposes.

---

## Prerequisites

No additional software is required.  
You only need:

- An up-to-date **web browser** (Chrome, Firefox, Edge, Safari, etc.).

---

## How to Open the Project

Since the project is built with **HTML**, **CSS**, and **JavaScript**, opening it is very simple:

1. Download or clone the repository.
2. Locate the main file:  
   **`index.html`**
3. Double-click the `index.html` file to open it in your browser.  
   *Alternatively, you can drag it directly into the browser window.*
4. That’s it! You can now browse the gallery and explore all sections of the site.

> No installations, servers, or dependencies are required.
